
### Used in **multipath / multi-controller environments**.

| SC   | Name                              |
| ---- | --------------------------------- |
| 0x00 | Internal Path Error               |
| 0x01 | Asymmetric Access Persistent Loss |
| 0x02 | Asymmetric Access Inaccessible    |
| 0x03 | Asymmetric Access Transition      |
| 0x04 | Controller Pathing Error          |
| 0x05 | Host Pathing Error                |
| 0x06 | Command Aborted by Host           |
